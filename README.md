# Weather_App
Flask and SqlAlchemy app

*Windows install:*

1. Run cmd
2. git clone https://github.com/Str1kez/Weather_App.git or go to downloaded folder
3. Run command : 
  <code> venv_windows\Scripts\python.exe app.py </code>
4. Go to http://127.0.0.1:5000/ in browser
